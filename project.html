CN Management System — Project Overview

A Progressive Web App (PWA) for automated Credit Note generation, PDF creation, email dispatch and ledger logging. Built with React, Firebase & Google Apps Script.

React Firebase Google Apps Script PWA jsPDF IndexedDB

Project Summary

Status: In production

This PWA automates the full lifecycle of Credit Notes for a business: create, preview, generate branded PDFs, save to Drive, email recipients, and log entries in a Google Sheet ledger. Built and deployed in a live environment.

Key Features

  • Interactive dashboard with KPIs and recent CN list
  • Guided Create CN form with templates and auto-calculations
  • Client-side PDF generation (party + printer copies)
  • Google Drive + Google Sheets orchestration via Apps Script
  • Auto email dispatch to Party, Head Office & Printer
  • Party management, bulk upload (XLSX), templates & audit logs
  • PWA caching & offline support with IndexedDB

Config Notes

Edit the following files to configure the app before use:
  • appscript_codename.txt — update sheet & folder IDs and company info (see lines in SETUP.md)
  • /services/googleScriptService.ts — set Apps Script URL
  • /services/firebaseService.ts — insert Firebase config
